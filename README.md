# Smart Invest Sandbox 

A **risk-free stock market simulation platform** that lets users start with **$100,000 virtual balance** to learn trading, practice investment strategies, and master financial markets without real financial risk.

##  Features

###  **Trading & Portfolio Management**
- **Virtual $100,000 Starting Balance** - Practice with realistic capital
- **Real-time Stock Trading** - Buy/sell stocks with live market data
- **Portfolio Tracking** - Monitor holdings, gains/losses, and performance
- **Transaction History** - Complete record of all trading activity
- **Diversification Analysis** - Portfolio health and risk assessment

###  **Educational System**
- **Interactive Tutorials** - Structured learning paths for all levels
- **Beginner to Advanced Courses** - From stock basics to professional strategies
- **Quizzes & Exercises** - Test your knowledge with interactive content
- **Progress Tracking** - Monitor your learning journey with badges and certifications

###  **AI-Powered Tools**
- **Market Forecasting** - LSTM neural network predictions for stock prices
- **AI Chatbot Assistant** - Get instant answers about stocks and portfolio
- **Trading Recommendations** - Data-driven investment suggestions
- **Risk Analysis** - AI-powered portfolio risk assessment

###  **Market Intelligence**
- **Live Market Data** - Real-time stock prices and market movements
- **Financial News Feed** - Curated news from multiple sources
- **Technical Analysis** - Chart patterns and technical indicators
- **Fundamental Analysis** - Company financials and valuation metrics

###  **User Experience**
- **Responsive Dashboard** - Beautiful, intuitive interface
- **Multi-Level Tutorials** - Progressive learning system
- **Virtual Certification** - Earn badges and certifications
- **Risk-Free Environment** - Learn without financial consequences

##  **Getting Started**

### **Prerequisites**
- Java 23 or higher
- Node.js 16+ 
- Maven 3.6+

### **Installation**

1. **Clone the repository**
```bash
git clone https://github.com/your-username/smart-invest.git
cd smart-invest
```

2. **Backend Setup**
```bash
cd src/main/java/backend
mvn clean compile
mvn exec:java -Dexec.mainClass="backend.Main"
```

3. **Frontend Setup**
```bash
cd src/frontend
npm install
npm run dev
```

4. **Access the Application**
- Frontend: `http://localhost:3000`
- Backend API: `http://localhost:8080`
